**Hospital Management System (C – Linked List Based)**

A simple console-based Hospital Management System written in C using a singly linked list for dynamic data storage and binary file handling for persistence.

**Features**

Add new patient

View all patients

Search patient by ID

Edit patient details

Delete patient record

Automatic file save & load (patients.dat)

Dynamic memory management using linked list

**Technologies Used**

C Programming

Linked List (Dynamic Data Structure)

File Handling (Binary File)

Standard Libraries (stdio.h, stdlib.h, string.h)

**Data Stored**

Each patient record contains:

ID (Auto Increment)

Name

Disease

Cabin Number

Phone Number

Age
